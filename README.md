A Japanese translation of the GPL
=================================

```
This is an unofficial translation of the GNU General Public License into
これはGNU 汎用公共ライセンス(GPL)の非公式な日本語訳である。

Japanese. It was not published by the Free Software Foundation, and does
これはフリーソフトウェア財団が発行したものではないので、

not legally state the distribution terms for software that uses the GNU
GNU GPL を用いるソフトウェアの配布条項を法的に述べるものではない

GPL--only the original English text of the GNU GPL does that. However,
――法的な効力があるのは、オリジナルの、英語の GNU GPL のみである。しかし

we hope that this translation will help Japanese speakers understand the
この翻訳が、日本語話者たちにとって GNU GPL をよりよく理解するための一助と

GNU GPL better.
なればよいと私たちは思っている。
```
